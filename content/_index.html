Why use Epinio ?

Epinio is designed for resource restricted environments.

Get Started

1. Download Epinio - latest release, x86_64, and ARM64 are supported on Linux, Mac, or Windows
2. Install Epinio 

epinio install

# ... cd to your application source ...
epinio push my_app .
