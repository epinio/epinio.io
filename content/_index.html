Why use Epinio ?

Simplicity

Epinio completely hides all Kubernetes and Container details from you. As a developer you can focus on the code. Epinio takes care of everything else.

DevOps

Epinio covers the complete application lifecylce and supports true DevOps. From your developer workstation to your scaled production cluster.

Lightweight

Epinio installs as a single Go binary and can be removed without leaving a trace.

Get Started

1. Download Epinio - latest release, x86_64, and ARM64 are supported on Linux, Mac, or Windows
2. Install Epinio 

epinio install

# ... cd to your application source ...
epinio push my_app .
